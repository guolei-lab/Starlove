# 部署与运行说明

## 1. 云开发环境

1. 打开微信开发者工具
2. 进入云开发控制台，创建环境
3. 记录环境 ID
4. 在 `miniprogram/app.js` 中设置 `globalData.cloudEnvId`

## 2. 部署云函数

云函数目录：`cloudfunctions/socialApi`

在微信开发者工具里：

1. 右键 `socialApi` -> "在终端中打开"
2. 执行 `npm install`
3. 右键 `socialApi` -> "上传并部署：云端安装依赖"

## 3. 创建数据库集合

在云数据库中创建以下集合（留空即可）：

- `users`
- `rooms`
- `messages`
- `friends`
- `blocks`
- `reports`

## 4. 推荐权限规则

建议每个集合权限都设置为：

- 仅创建者可读（按需）
- 禁止客户端直接写入
- 所有写操作通过云函数

如果你后续使用自定义规则 JSON，可按“仅云函数写、按 openid 隔离读”策略扩展。

## 5. 本地运行验证

1. 编译小程序
2. 授权登录
3. 开始匹配
4. 进入聊天发送消息
5. 倒计时结束后添加好友
6. 个人中心查看历史会话

## 6. 常见问题

- 提示 `Function not found`：未部署 `socialApi`
- 提示 `服务异常`：检查云函数日志
- 匹配不到人：当前环境在线用户太少，需至少 2 个测试账号
- 头像空白：请确认授权返回了头像，或设置默认头像资源
